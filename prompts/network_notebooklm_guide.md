# Google NotebookLM 활용 가이드 — 네트워크 기초 편

> NotebookLM: https://notebooklm.google.com
> 목적: 온프레미스 네트워크 기초 교육 자료를 NotebookLM으로 학습 강화

---

## Step 1 — 소스 자료 업로드

NotebookLM 접속 → 새 노트북 생성 후 다음 중 하나 이상 추가:

- `edu_network_basics.md` 파일을 복사해서 "텍스트 붙여넣기"로 추가
- 추가 PDF 자료가 있으면 함께 업로드

---

## Step 2 — 학습 가이드 생성 프롬프트

소스 추가 후 채팅창에 아래 프롬프트 입력:

```
업로드한 자료를 바탕으로 IT 비전공 직장인을 위한 네트워크 기초 학습 가이드를 만들어주세요.

다음 형식으로 출력해주세요:

1. 핵심 개념 요약 (10가지, 각각 2~3줄)
2. 비전공자를 위한 쉬운 비유 목록 (각 개념당 비유 1개)
3. 자주 묻는 질문 5가지와 답변
4. 실전 활용 시나리오 3가지 (업무에서 네트워크 지식이 필요한 상황)
5. 다음 학습 단계 추천 로드맵 (네트워크 → 클라우드 → 보안)
```

---

## Step 3 — 퀴즈 생성 프롬프트

```
이 자료로 복습 퀴즈 10문제를 만들어주세요.
- 난이도: 초급 (IT 비전공자 기준)
- 형식: 4지선다 7문제 + OX 3문제
- 각 문제에 정답과 해설 포함
- 실생활 예시로 문제를 구성해주세요 (포트 번호, 상태코드, 방화벽 등)
```

---

## Step 4 — 오디오 다이제스트 (팟캐스트)

NotebookLM 우측 상단 **"오디오 개요"** 버튼 클릭
→ 자동으로 2인 대화 형식 팟캐스트 생성 (약 8~12분)
→ 출퇴근 길에 듣기용으로 활용

---

## 추천 학습 흐름

```
교육 자료 읽기 (15분)
        ↓
NotebookLM 학습 가이드 생성 (10분)
        ↓
퀴즈로 복습 (10분)
        ↓
Claude 앱에서 PPTX 생성 → 발표 준비 (30분)
        ↓
팟캐스트 반복 청취로 장기 기억화
```

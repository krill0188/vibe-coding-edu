```
아래 내용을 바탕으로 PowerPoint(PPTX) 프레젠테이션 구조를 만들어주세요.

[조건]
- 대상: IT 비전공자, AI 코딩 도구를 처음 접하는 직장인
- 슬라이드 수: 15~18장
- 언어: 한국어
- 스타일: 심플하고 깔끔, 각 슬라이드에 핵심 한 줄 + 불릿 3개 이내
- 비유와 예시를 반드시 포함
- 마지막 슬라이드는 "오늘 배운 핵심 10가지" 요약표로 마무리

[슬라이드 구성]
1. 표지: "바이브 코딩을 위한 필수 IT 지식 10가지"
2. 바이브 코딩이란? (정의 + 왜 지금인가)
3. AI 도구 비교: ChatGPT vs Claude vs Cursor
4. 작업 환경: CLI vs IDE (주방 비유)
5. 코드가 앱이 되는 과정: 소스 → 빌드 → 배포
6. Git이란? (버전 관리의 의미)
7. Git 3가지 동작: commit / push / pull
8. 프론트엔드 vs 백엔드 (카페 홀 vs 주방)
9. 포트란? (방 번호 비유, 주요 포트 번호 표)
10. API란? (식당 주문 시스템 비유)
11. HTTP 메서드 4가지: GET / POST / PUT / DELETE
12. JSON이란? (데이터 전달 형식)
13. 데이터베이스: SQL vs NoSQL (엑셀 vs 서랍장)
14. 배포 서비스 비교: Vercel / Cloudflare / AWS
15. 바이브 코딩 실전 팁 3가지
16. 오늘 배운 핵심 10가지 요약표
17. Q&A / 마무리

[출력 형식 — 슬라이드마다 아래 구조로]
---
슬라이드 N: [제목]
헤드라인: (한 줄 핵심 메시지)
내용:
• 불릿 1
• 불릿 2
• 불릿 3
발표자 노트: (발표 시 말할 내용 2~3문장)
---
```

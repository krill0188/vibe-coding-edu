아래 내용을 바탕으로 PowerPoint(PPTX) 프레젠테이션 구조를 만들어주세요.

[조건]
- 대상: IT 비전공자, 네트워크 개념이 처음인 직장인
- 슬라이드 수: 15~18장
- 언어: 한국어
- 스타일: 심플하고 깔끔, 각 슬라이드에 핵심 한 줄 + 불릿 3개 이내
- 비유와 예시를 반드시 포함
- 마지막 슬라이드는 "오늘 배운 핵심 10가지" 요약표로 마무리

[슬라이드 구성]
1. 표지: "온프레미스 네트워크 기초 완전 정복"
2. 이 자료로 무엇을 알게 되는가? (클라이언트→서버 여정 10개 개념)
3. 인터넷 vs 웹: 도로망(인터넷) vs 자동차(웹)
4. IP 주소: 컴퓨터의 우편 주소 (IPv4 vs IPv6)
5. TCP: 데이터 분할 배송 시스템 (택배 비유)
6. TCP의 3가지 보장: 분할 전송 / 순서 보장 / 재전송
7. HTTP: 웹의 주문서 양식 (요청 / 응답 / 상태코드)
8. HTTPS와 자물쇠 아이콘의 의미
9. 라우터: 교차로 신호등 (경로 결정)
10. 스위치: 사무실 내선 교환기 (LAN vs WAN)
11. 방화벽: 건물 보안 게이트 (허용 포트 표)
12. CDN: 전국 편의점 물류 거점 (콘텐츠 분산 배포)
13. DMZ: 외부 손님 대기실 (내부망 격리 구조도)
14. Web / WAS / DB 3계층 아키텍처
15. 이중화(HA): 주방 2개 + 금고 이중 잠금
16. 핵심 10가지 요약표 + 실전 팁 3가지
17. Q&A / 마무리
